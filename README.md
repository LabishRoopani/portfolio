# Labish Roopani — Portfolio

Personal portfolio website built with React + Vite.

## Stack
- React 18
- Vite
- Plain CSS 

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

Push this repo to GitHub, then import it at vercel.com/new.
Vercel auto-detects Vite — no extra config needed.
